# Maggie XLM-R Embeddings

Files in this directory are generated by the Maggie-first pipeline.

Required metadata:

- `unique_text_metadata.csv`
- `row_embedding_metadata.csv`
- `embedding_config.json`

Embedding arrays:

- `unique_text_embeddings.npz` is stored directly.
- `row_embeddings.npz` is stored directly.

Validate reconstructed shards with the corresponding `.sha256` files.
